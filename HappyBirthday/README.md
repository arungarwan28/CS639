# Personalized Birthday Card App

This Android app, built with Jetpack Compose, displays a personalized birthday card with a custom background image and a unique greeting message. The app showcases the use of modern Android development techniques to create a visually appealing and customizable user interface.

## Features

*   **Custom Background Image:** The app uses a personalized background image to create a unique and visually appealing design.
*   **Dynamic Greeting:** The birthday message is dynamically generated to include the recipient's name and a personalized message from the sender.
*   **Modern UI:** The user interface is built entirely with Jetpack Compose, showcasing a modern approach to Android UI development.
*   **Clean Architecture:** The code is structured into reusable composable functions, promoting a clean and maintainable codebase.

## Customization

To customize the birthday card, you can modify the following:

*   **Recipient's Name:** Change the `recipient` parameter in the `BirthdayGreetingWithImage` composable.
*   **Sender's Name:** Update the `sender` parameter to change the name of the person sending the card.
*   **Background Image:** Replace the `img.png` file in the `res/drawable` folder with your desired background image.
